# Xigua-Music 西瓜音乐
一个基于DuerOS SDK开发的智能音箱应用

## 如何使用

1. 执行 `composer install` 安装DuerOS的bot-sdk
2. 部署到你的server上
3. 根据[DuerOS 开发文档](https://dueros.baidu.com/dbp)进行部署
4. 测试效果

## 目前实现功能

1. 播放热门歌曲
2. 搜索指定歌手的指定歌曲
3. 下一首
4. 歌词显示
5. 播放完毕之后，自动下一首
